Introduction

Get started with Animation.css and learn by exploring interactive examples.
Animations can greatly improve an interface's UX, but it's important to follow some guidelines to not overdo it and deteriorate the user experience on your web-things. Following the following rules should provide a good start.

Installation and Usage

Install with npm:

$ npm install VTanimate.css --save

Import it into your file:

import 'VTanimate.css';

You can added it directly to your webpage using a CDN:

< head >
< link rel="stylesheet" Link" />
< / head>

After installing Animate.css, add the any class of the animation names.

< h1 class="bounce">An animated element< / h1>
